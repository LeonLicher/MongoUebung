# Database Performance Comparison

A minimal web application comparing Redis and MongoDB performance.

## Quick Start

```bash
docker-compose up --build
```

Open http://localhost:3000 and click "Run Benchmark".

## Requirements

- Docker
- Docker Compose

That's it. No Node.js or other dependencies needed on your machine.

## What it does

Compares 1,000 write and read operations between Redis and MongoDB.
